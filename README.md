## <div align="center">streamlit による株価分析 app </div>

- Python version 3.11.7

- 仮想環境作成<br>
  => python -m venv env_st_stock

- 仮想環境起動<br>
  => .\/env_st_stock/Scripts/activate

- requirements.txt<br>
  pip install -r requirements.txt<br>
  pip freeze > requirements.txt

#### アプリの実行

- streamlit run main.py
